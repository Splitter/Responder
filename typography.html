Paragraphs

A paragraph is a distinct section of a piece of writing. There are 2 types of dropcaps built into the theme. Adding 'dropcap' or 'dropcap2' to the class of a paragraph( p ) tag will add a dropcap to it. 

<p class="dropcap">
	This is the body of the paragraph...
</p>

Pull Quotes

Pullquotes, not to be confused with blockquotes, also symbolizes a quote of a block of text. The difference lies in the visual presentation as well as the content. PullQuotes are most often used in magazine/news articles and pull the quotes directly from the text of the article it is placed in and tend to be floated in the content of the article. Pullquotes are meant to help engage the viewer in the article more quickly by displaying an interesting 'teaser' of the article content. Pullquotes have no distinct html tags associated with them, this theme uses h6 headings for them to help keep a semantic mark-up 

<h6 class="pullright OptionalClass">
	This is the pullquote text...
</h6>

Code and Pre Tags

Code tags are used to highlight text that represents code, usually within a body of text. Pre tags are used to display preformated bodies of text, where whitespace is preserved. Pre tags are most commonly used to display a larger block of code preserving nesting and tabbing to make it easier to read.

Example: 
<p>
This text is contained within a <code> p </code> element and if displayed the 'p' would be highlighted.
</p>

<pre>
	<div>
	This text is in a div tag which is within a pre tag.
	</div>
</pre>
